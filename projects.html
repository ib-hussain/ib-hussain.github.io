Cinemago

Movie recommendation site.

Cinemago is a movie discovery and rating platform built using Flask. Users can browse, rate, and suggest movies. Admins can add new content via web scraping, and user preferences are stored in a lightweight SQLite database.

Network Emulator

The Network Emulator simulates a computer network.

The computer network consisting of routers and end machines (e.g., computers).
Messages are sent from one device to another using destination addresses and follow a routing path across multiple routers. This project aims to simulate routing, queuing, and message handling in a simplified network.

Conway's Game of Life

Cellular automaton simulation

An implementation of John Conway's Game of Life, a zero-player game that simulates cellular evolution based on simple rules. The simulation runs on a 2D grid where each cell can be alive or dead, evolving through generations. Features include adjustable grid size, speed control, and pattern saving/loading.

Github Repository:
https://github.com/ib-hussain/Game-Of-Life
